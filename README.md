osm4u
=====

OSM-based POI service
Сервис по ПОИ на основе ОСМ.

Структура

www - морда сайта.

www/api - интерфейс к базе. Возращает и принимает xml для каждой операции.


POIProcessor - загрузчик пои в базу.
               работает над польским файлом. (VB6)

winpt  - прототип пользовательского интерфейса, для windows.
	хотя это виндовое приложение, данные для каждой операции получаются с web, через апи (www/api)
